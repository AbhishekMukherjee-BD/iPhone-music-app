1. ARCHITECTURE OVERVIEW & IMPLEMENTATION DETAILS
------------------------------------------------------------------------
Noir Sound is built as an offline-first Progressive Web App (PWA). This bypasses the need for the full Xcode development suite while providing a high-fidelity, native-feeling music player on iOS.

Core Technical Blocks:
* Offline Caching (Service Worker): `sw.js` caches all core player assets (index.html, app.css, app.js, manifest.json, local icons, and the jsmediatags CDN script) so the app launches instantly offline.
* Local Database (IndexedDB): App uses a custom IndexedDB database (`NoirSoundDB`, store `songs`) in `app.js` to store MP3 audio files as raw binary Blobs. This ensures imported music is persistently stored on the device and remains playable without network access.
* Metadata Extraction & Fallback: Reads MP3 ID3 tags (title, artist, album art) using the `jsmediatags` library. Convert raw cover art bytes to Base64 strings for persistent storage in IndexedDB. If metadata is missing or fails to parse, it falls back to parsing the filename (e.g. splits "Artist - SongName.mp3" safely).
* Background Play & Lock Screen (Media Session API): Configured standard HTML5 `<audio>` playback. When playing, it hooks into the native iOS Media Session API, syncing the track title, artist, and cover art to the iOS Lock Screen & Control Center, and routes playback controls (play, pause, skip, prev, seek slider) back to JS event handlers.
* iOS Gesture Interactions: Implemented touch listeners on the player's drag handle so users can swipe/drag down to dismiss the full-screen player panel.
* Haptic Button Animations: Tapping controls triggers scale-down animations (`scale(0.92)`) with spring easings for satisfying physical feedback.

2. TWO INTERFACES IMPLEMENTED
------------------------------------------------------------------------
We have built two distinct visual interfaces switchable in real-time using the theme toggle button (id="theme-toggle" inside header):

- Theme 1: Luxury Obsidian & Gold (Default)
  - Color palette: Obsidian charcoal gradients, metallic gold accents, gold text.
  - Cards: Semi-transparent glassmorphic player card with gold borders, radial gold backing glows.
  - Special Animations: Album art is styled as a CD (circle) that rotates dynamically while playing, and features a glowing gold drop-shadow that pulses with the audio.
  
- Theme 2: Apple Minimalist
  - Color palette: Deep stark black background, flat dark-gray blocks, minimal white/gray text.
  - Accent color: Apple Music pink/red (`#ff2d55`) for active buttons.
  - Cards: Sharp, flat cards with iOS 12px rounded corners, crisp borders, and zero glowing elements.
  - Font: Clean, bold San Francisco-style sans-serif typography.
