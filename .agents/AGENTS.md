# Project-Scoped Rules

## Git Push & Auto-Deployment Guidelines
* **Prompt for Upload:** Whenever any workspace changes are made (HTML, CSS, JS, etc.), you must ask the user for permission before pushing the changes to the repository.
* **Commit and Push:** If the user approves, run the staging, commit, and push operations to the remote repository (`https://github.com/AbhishekMukherjee-BD/iPhone-music-app.git`) on the `main` branch to trigger Vercel's CD build.
